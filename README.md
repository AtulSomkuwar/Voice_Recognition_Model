# Voice_Recognition_Model


Collected Voices of different peoples given to MfCC to extract the features from everyone voices later on RandomForest model is applied to train the model at last PCA is applied to make a grouping of each voice samples 

Steps Followed
1) Collected Voice recording of 9 peoples each has 10 samples
2) Converted WAV file into mp4
3) MFCC Feature Extraction 
4) Shuffling the data for not to leasrning the pattern from the dataset
5) Feature Scaling 
6) Trained a Model using 80% training and 20% testing
7) PCA
